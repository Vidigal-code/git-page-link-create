# Funcao QR

As ferramentas de QR ficam disponiveis no `/create` em modo QR.

## Capacidades

- Gerar QR a partir de texto ou URL
- Configurar tamanho, margem e nivel de correcao
- Exportar PNG e SVG
- Copiar imagem QR e abrir em popup/janela

## Integracao com links

- Pode gerar links de render para `/r` e `/ra`
- Pode usar short URL automaticamente para links muito longos
