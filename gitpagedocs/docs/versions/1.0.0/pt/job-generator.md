# Gerador de Vagas de Emprego (Jobs)

A ferramenta contida sob o esquema de navegação `create-jobs` em `/pages` promove recursos diretos de marcação estruturada e documentação dinâmica para recrutadores e empresas gerarem páginas atrativas relativas a contratações.

## Por que utilizar este módulo?
Na plataforma do **GitPageLinkCreate**, compartilhar propostas de emprego se torna um formato independente das vias convencionais limitadas de mercado (LinkedIn clássico). 

Você utiliza o formulário contido na página de criação que armazena nativamente:
* **Escopo e Título:** Ex: "Sr Backend Engineer".
* **Contratação e Salário:** Faixa e formato (CLT / PJ).
* **Stack Tecnológico:** Linguagens base (Node, .Net, AWS).
* **Botão CTA:** O formulário principal de coleta ou link de e-mail integrado para envio automático do currículo.

## Saída do Gerador
Ao criar um conteúdo de página, o software processa todos esses JSONs provisórios em uma página pré-renderizada pela plataforma (ou link comprimido e traduzido). 
Esta página tem tipologia Otimizada para SEO e é responsiva.

A partir desse local de destino (*render_job_page*), as chamadas para submissão redirecionam perfeitamente sem o usuário necessitar recarregar ou sofrer requisições engasgadas, favorecendo candidaturas fluidas e amigáveis a dispositivos móveis.
