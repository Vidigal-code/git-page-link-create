# Funcion QR

Las herramientas QR estan disponibles en `/create` en modo QR.

## Capacidades

- Generar QR desde texto o URL
- Configurar tamano, margen y nivel de correccion
- Exportar PNG y SVG
- Copiar imagen QR y abrir en popup/ventana

## Integracion con enlaces

- Puede generar enlaces render para `/r` y `/ra`
- Puede usar short URL automaticamente para enlaces muy largos
